# 接下来只需在微信开发者工具完成的操作

## 1. 导入和编译

1. 安装微信开发者工具稳定版。
2. 导入 `D:\Projects\wechat-meeting-minutes`。
3. 使用小程序管理员微信扫码登录。
4. 确认 AppID 为 `wx9ed761b2e83810a0`。
5. 点击“编译”，先处理开发者工具显示的任何基础库或隐私声明提示。

## 2. 确认云环境

- 环境 ID：`cloud1-d1gn26yde8fd1f267`
- 确保开发者工具右上角选中的云环境与该 ID 一致。

## 3. 初始化数据库

1. 右键 `cloudfunctions/setup`。
2. 选择“上传并部署：云端安装依赖”。
3. 在云开发控制台进入云函数 `setup`，点击“云端测试”。
4. 返回结果应包含5个集合：`meetings`、`audioSegments`、`processingJobs`、`providerConfigs`、`invitedUsers`。
5. 依次打开5个集合的“权限设置”，每个集合都应用项目根目录的 `database.rules.json`，禁止客户端直接读写。

## 4. 配置云函数密钥

为 `api` 和 `processor` 配置相同的环境变量：

```text
CONFIG_ENCRYPTION_KEY=至少32位随机字符串
INVITE_MODE=off
```

首次联调保持 `INVITE_MODE=off`。确认功能正常后再打开白名单。

## 5. 部署业务函数

1. 右键 `cloudfunctions/api`，选择“上传并部署：云端安装依赖”。
2. 右键 `cloudfunctions/processor`，选择“上传并部署：云端安装依赖”。
3. 确认 `processor` 已创建每分钟执行一次的定时触发器。

## 6. 真机联调

1. 打开小程序“接口设置”，先配置并测试转写和总结接口。
2. 录制30秒测试音频并结束。
3. 确认状态依次变为等待处理、识别中、总结中、已完成。
4. 确认首页显示一句话主旨，详情页显示纪要并可播放录音。

## 7. 体验版

1. 在公众平台填写用户隐私保护指引。
2. 添加体验成员。
3. 在开发者工具上传代码。
4. 在公众平台将开发版本设为体验版并分享体验二维码。

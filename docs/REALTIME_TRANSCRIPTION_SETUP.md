# 实时转写部署

## 1. 微信小程序后台

在小程序后台的“开发管理 → 开发设置 → 服务器域名”中，添加 Socket 合法域名：

```text
wss://asr.cloud.tencent.com
```

真机不添加该域名不能建立实时转写连接。开发者工具中关闭“校验合法域名”仅适用于本地调试，不能替代这一步。

## 2. 小程序设置页

选择“腾讯云实时转写 + 异步回退”，填写并保存：

- 腾讯云 AppID（腾讯云 API 密钥管理页中的 AppID，不是微信小程序 AppID）
- API Key（腾讯云 SecretId）
- SecretKey（腾讯云 SecretKey）
- 地域 `ap-guangzhou`
- 模型 `16k_zh`

腾讯云密钥只会加密存入云端；小程序只获得由云函数签发、15 分钟有效的单次 WebSocket URL。

## 3. 部署

在微信开发者工具中分别右键并选择“上传并部署：云端安装依赖”：

- `cloudfunctions/api`
- `cloudfunctions/processor`

然后重新编译小程序并真机调试。

## 4. 预期流程

开始录音后，录音页“实时逐字稿”应在数秒内显示稳定文本。点击完成后，实时逐字稿直接交给 DeepSeek 生成纪要；WebSocket 失败、暂停或网络中断时，录音仍会保存，并自动回退到录后异步识别。
